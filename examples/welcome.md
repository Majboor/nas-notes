# Welcome

This is a **Cherry** note — just a Markdown file in your notes folder
(`$NAS_NOTES_DIR`, default `~/.rpidrive/notes`).

## How it works
- The first `# Heading` line becomes the note's **title** (and its Spotlight app name).
- Everything else is rendered as Markdown when you read the note.
- Hit **Edit** to change it, **Save** to write it back to disk.

## Try it
- Click **+** on the All Notes screen to add a note.
- Press ⌘-Space and type this note's name to jump straight here.

```bash
# Notes are plain files — grep, back up, or sync them however you like.
ls "$NAS_NOTES_DIR"
```

> Tip: put anything you re-use a lot here — commands, checklists, handover prompts —
> then Copy it in one click.
